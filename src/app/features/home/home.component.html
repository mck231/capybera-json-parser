Capybera Json Tool

Why

Trying to read Json data shouldn't be distracting or dificult. Editing Json data should be simple and result in valid Json.

What

This is a static web application written using the Angular framework. Our tool lets our users upload, visualize, edit, and export json data.

Who

Intended audience is anyone working with Json and needs a more robust tool to work with data.

Fast and easy way to create mock objects or test structures.
